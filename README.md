# Skillfolio

Skillfolio is a personal resume-sharing portfolio website designed to showcase multiple role-specific resumes in a clean, modern, and fullscreen viewing experience.

This project allows visitors to select a specific profile (AI/ML, Data Science, or Data Analytics) and view the corresponding resume seamlessly.
